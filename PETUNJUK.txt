CARA PAKAI
==========
Struktur folder:
  index.html        -> halaman utama
  css/style.css      -> semua styling/tampilan
  js/script.js        -> semua logika (countdown, musik, galeri foto)
  assets/photos/    -> taruh foto kamu di sini
  assets/music/      -> taruh musik kamu di sini

1. FOTO
   - Taruh foto-foto kamu di folder: assets/photos/
   - Beri nama: foto1.jpg, foto2.jpg, foto3.jpg, dst.
   - Default sekarang sudah diset untuk format .png
     (foto3.png contoh sudah dimasukkan).
     Kalau fotomu formatnya .jpg atau .jpeg, buka js/script.js,
     cari baris:
       const PHOTO_EXT = 'png';
     ganti jadi 'jpg' atau 'jpeg' (harus SAMA semua untuk semua
     foto, jangan dicampur format-nya).
   - Kalau jumlah foto kamu lebih/kurang dari 6, cari baris:
       const TOTAL_PHOTOS = 6;
     ganti sesuai jumlah foto kamu.

2. MUSIK
   - Taruh 1 file musik di folder: assets/music/
   - Beri nama file: song.mp3
   - (Kalau mau pakai format lain seperti .ogg, ganti juga
     bagian <source src="assets/music/song.mp3" ...> di index.html)

3. TANGGAL COUNTDOWN
   - Sudah diatur ke 1 Juli 2026, 00:00.
   - Kalau mau ganti, buka js/script.js, cari baris:
       const targetDate = new Date("2026-07-01T00:00:00");
     lalu ganti tanggalnya.

4. KATA-KATA / SURAT
   - Buka index.html, cari bagian "Sepucuk Surat Untukmu"
   - Edit teks di dalam tag <p> sesuai kata-kata kamu sendiri.

5. WARNA / TAMPILAN
   - Semua warna diatur di css/style.css bagian paling atas (:root),
     misalnya --sky-deep, --sky-mid, --center-yellow, dst.
     Tinggal ganti kode warnanya kalau mau ubah nuansa biru/daisy.

6. CARA BUKA WEBSITE
   - Cukup double click file index.html, akan terbuka di browser.
   - Pastikan folder css/, js/, dan assets/ tetap satu folder
     dengan index.html (jangan dipisah), karena saling terhubung.
   - Sebelum tanggal 1 Juli 2026 00:00, halaman menampilkan countdown.
     Setelah waktu itu lewat, seluruh isi spesial otomatis terbuka.

CATATAN SOAL AUTOPLAY MUSIK
----------------------------
Browser modern (Chrome, Safari, dst) biasanya memblokir audio
autoplay dengan suara sebelum ada interaksi dari user (klik/tap).
Script sudah diatur supaya musik otomatis coba diputar begitu
konten terbuka, dan kalau diblokir, otomatis akan mulai diputar
begitu pacar kamu klik/tap pertama kali di halaman (misalnya saat
scroll). Jadi musik tetap akan terdengar tanpa dia perlu mencari
tombol play.
